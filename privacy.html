Kebijakan Privasi

Transparansi adalah fondasi kepercayaan. Kami berkomitmen melindungi privasi data Anda.

Pengumpulan Data

FaraidhKu adalah aplikasi web Client-Side. Artinya, semua proses perhitungan waris terjadi di dalam browser perangkat Anda sendiri.

  • Kami tidak menyimpan data input harta, keluarga, atau ahli waris Anda di server kami.
  • Kami tidak mengirimkan data pribadi Anda ke pihak ketiga manapun.
  • Data Anda hilang segera setelah Anda menutup tab browser atau melakukan refresh halaman.

Penggunaan Cookies

Saat ini, FaraidhKu tidak menggunakan cookies pelacak atau analitik pihak ketiga. Kami fokus pada fungsionalitas murni tanpa memantau perilaku pengguna.

Keamanan

Karena tidak ada data yang dikirim ke server (backend), risiko kebocoran data sangat minim. Keamanan data bergantung pada keamanan perangkat dan browser yang Anda gunakan.
